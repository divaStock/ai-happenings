USAGE EXAMPLES
================================================================================

1. STANDALONE EXECUTION
   $ cd src
   $ python main.py
   
   Output: Complete analysis with top articles and LinkedIn posts

2. n8n WEBHOOK SERVER
   $ cd src
   $ python n8n_webhook.py
   
   Server: http://localhost:5000

3. API CALL (Complete Pipeline)
   $ curl -X POST http://localhost:5000/run-pipeline \
     -H "Content-Type: application/json" \
     -d '{
       "agentbill_api_key": "your-key",
       "openai_api_key": "your-key",
       "customer_id": "production"
     }'

4. API CALL (Top Posts Only)
   $ curl -X POST http://localhost:5000/get-top-posts \
     -H "Content-Type: application/json" \
     -d '{"count": 5, "agentbill_api_key": "...", "openai_api_key": "..."}'

CUSTOMIZATION GUIDE
================================================================================

1. Add News Sources
   Edit: src/scrapers/web_scraper.py
   Add to self.sources list

2. Change Keywords
   Edit: src/scrapers/web_scraper.py
   Modify self.keywords list

3. Adjust AI Model
   Edit: src/analyzers/ai_analyzer.py
   Change model="gpt-4" to "gpt-3.5-turbo"

4. Modify Prioritization Weights
   Edit: src/prioritizers/article_prioritizer.py
   Adjust self.weights dictionary

5. Add Custom Tracking Events
   Call: self.agentbill.track_signal(event_name, revenue, data)
